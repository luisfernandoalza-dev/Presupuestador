# Estudio A · Generador de Presupuestos

Repositorio listo para subir a GitHub Pages o a cualquier hosting estático.

## Estructura

- `index.html`: aplicación principal
- `assets/css/styles.css`: estilos generales, documento y print A4
- `assets/js/app.js`: lógica de edición, render y exportación
- `assets/img/brand/logo-estudio-a.png`: logo institucional
- `robots.txt`, `sitemap.xml`, `site.webmanifest`: base SEO

## Ajustes pendientes recomendados

- Reemplazar `https://example.com/` por la URL final del dominio o GitHub Pages.
- Si querés SEO cerrado al uso interno, cambiar `robots.txt` a `Disallow: /`.
- Si el logo final se reemplaza por versión optimizada SVG/PNG, mantener el mismo nombre de archivo o actualizar rutas.
